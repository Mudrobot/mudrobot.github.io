【论文笔记】DiP--Learning Discriminative Implicit Parts for Person Re-Identification

  • Authors: [[Dengjie Li]], [[Siyu Chen]], [[Yujie Zhong]], [[Lin Ma]]

  • Tags: #Computer-Science—Computer-Vision-and-Pattern-Recognition

文章内容

Abstract

In person re-identiﬁcation (ReID) tasks, many works explore the learning of part features to improve the performance over global image features. Existing methods explicitly extract part features by either using a hand-designed image division or keypoints obtained with external visual systems. In this work, we propose to learn Discriminative implicit Parts (DiPs) which are decoupled from explicit body parts. Therefore, DiPs can learn to extract any discriminative features that can beneﬁt in distinguishing identities, which is beyond predeﬁned body parts (such as accessories). Moreover, we propose a novel implicit position to give a geometric interpretation for each DiP. The implicit position can also serve as a learning signal to encourage DiPs to be more position-equivariant with the identity in the image. Lastly, an additional DiP weighting is introduced to handle the invisible or occluded situation and further improve the feature representation of DiPs. Extensive experiments show that the proposed method achieves state-of-the art performance on multiple person ReID benchmarks.

总结

核心思想

  • propose to learn Discriminative implicit Parts (DiPs)

  • propose a novel implicit position

  • introduced an additional DiP weighting

  • We propose to learn Discriminative implicit Parts (DiPs) by extracting features that are beyond explicit body parts and can significantly improve the ReID performance.

  • We propose an implicit position to give a geometric interpretation for each DiP in the image space. We also encourage the DiPs to learn to predict their implicit positions such that DiPs become position-equivariant with identities.

  • We further model the contribution of DiPs in images via the DiP weighting, and a part-based distance computation to enable a more fine-grained comparison between images.

DiPs

作用

通过提取有助于身份识别的与身体无关的特征来学习判别性隐式部分（简称 DiP）。

实现过程

整个方法的架构如下图所示：

Step 1 Image Division

将图片划分为像素为(P,P)(P,P)(P,P),设置划分步长为SSS,则最后的块数N为：

N=NH×NW=⌊H+S−PS⌋×⌊W+S−PS⌋N = N_H\times N_W = \lfloor \frac{H+S-P}{S} \rfloor\times\lfloor\frac{W+S-P}{S} \rfloor N=NH​×NW​=⌊SH+S−P​⌋×⌊SW+S−P​⌋

划分完成后，拍平成1D Tensor。然后经过Linear Projection 模块变成N个D维embeddings。

然后在我们得到的这个长度为N的Embedding特征队列的前面，我们还要加上一个长度D维CLS特征，和普通ViT的全局特征一样，并且我们在每个ID embedding前都要加上一个位置编码

Step 2 Discriminative Implicit Parts

目标： 使用DiPs 来预测隐藏的位置

实现方法就是简单的向ViT的输入部分加入了一些可以被学习的Tokens。（同样也加入了Position Encoding用于进行区分）

因此ViT的输入可以被表示为。

Z0=[ecls;ex1;ex2;⋯ ;exN;ed1;ed2;⋯ ;edM]+PZ_0 = [e_{cls};e_x^1;e_x^2;\cdots;e_x^N;e_d^1;e_d^2;\cdots;e_d^M]+P Z0​=[ecls​;ex1​;ex2​;⋯;exN​;ed1​;ed2​;⋯;edM​]+P

经过L层后，输出Transformer的输出我们记为：

ZL=[fcls;f1;f2;⋯ ;fN;dip1;dip2;⋯ ;dipM]Z_L = [f_{cls};f^1;f^2;\cdots;f^N;dip^1;dip^2;\cdots;dip^M] ZL​=[fcls​;f1;f2;⋯;fN;dip1;dip2;⋯;dipM]

隐藏位置预测(作为一种学习信号来训练MLP)

对于每一个dip，都将他与每个patch的向量计算相似度，最终得到一个相似度矩阵c，相似度矩阵c过一个softmax后，对每个位置求加权平均，最后就可以得到一个位置点，这个位置就是每一个dip的目标隐藏位置。

预测位置采用的是3层MLP结构，损失函数就是和计算目标位置的欧式距离。

Step 3 DiP Weighting

在某些图像中，一个或多个 DiP 可能不可见或被遮挡。在这种情况下，不可见或被遮挡的 DiP 对图像之间的比较的贡献应该较小甚至为零。

因此我们对DiP引入权重，让模型在训练和推理的时候能够动态调整每个DiP的贡献。

其实这个权重就是在后面接的MLP中和DiP位置预测一起被训练了。都会参与后面ID Loss和Triplet Loss 的计算

Step 4 encode each frame

接下来就是ReID问题常见的两个loss计算了

每张图片的编码我们采用所有DiP的加权和来进行编码：

dip^=∑iMωi∗dipi\hat{dip}=\sum_{i}^{M}\omega^i*dip^i dip^​=i∑M​ωi∗dipi

Triplet Loss 优化

采取的方法是，首先计算两张图对应DiP的欧式距离，记为ddd

然后根据两个图片对DiP的加权，分别记为w1,w2w_1,w_2w1​,w2​。计算一个加权距离：

dist=∑Softmax(w1∗w2)∗ddist = \sum Softmax(w_1*w_2)*d dist=∑Softmax(w1​∗w2​)∗d
