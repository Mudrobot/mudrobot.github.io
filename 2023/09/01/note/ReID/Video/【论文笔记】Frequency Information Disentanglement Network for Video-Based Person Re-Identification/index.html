Abstract

在视频人物重新识别（Re-ID）中，网络必须从连续帧中持续地提取目标人物的特征。现有的方法往往只关注如何使用时间信息，这常常导致网络被相似的外观和相同的背景所愚弄。

在本文中，我们提出了一种解缠结和交换聚合网络（DSANet），它将表示身份的特征和基于相机特征的特征分离，并且更加关注ID信息。

我们还引入了一项辅助任务，该任务利用通过交换和聚合创建的一对新feature来提高网络针对各种相机场景的能力。

此外，我们还设计了一个目标定位模块(TLM) ，该模块根据帧流提取目标位置变化时的鲁棒特征，以及一个反映最终表示中时间信息的帧权重生成(FWG)。

此外，我们还设计了一个目标定位模块(TLM) ，该模块根据帧流（frame flow） 以及一个反映最终表示中时间信息的帧权重生成(FWG) ，来提取目标位置变化时的鲁棒特征。

我们也设计了多种用于解纠缠学习的损耗函数，使网络的各个组成部分能够协同工作，同时满意地发挥各自的作用。
大量实验的定量和定性结果表明，DSANet 在三个基准数据集上优于最先进的方法。

In video person re-identification (Re-ID), the network must consistently extract features of the target person from successive frames. Existing methods tend to focus only on how to use temporal information, which often leads to networks being fooled by similar appearances and same backgrounds. In this paper, we propose a Disentanglement and Switching and Aggregation Network (DSANet), which segregates the features representing identity and features based on camera characteristics, and pays more attention to ID information. We also introduce an auxiliary task that utilizes a new pair of features created through switching and aggregation to increase the network’s capability for various camera scenarios. Furthermore, we devise a Target Localization Module (TLM) that extracts robust features against a change in the position of the target according to the frame flow and a Frame Weight Generation (FWG) that reflects temporal information in the final representation. Various loss functions for [[Disentanglement Learning]] are designed so that each component of the network can cooperate while satisfactorily performing its own role. Quantitative and qualitative results from extensive experiments demonstrate the superiority of DSANet over state-of-the-art methods on three benchmark datasets.

1. Introduction

人员重新识别(ReID),旨在从不同位置和角度匹配摄像机上特定人员的身份。由于智能监视系统和各种多媒体应用的扩散，对 Re-ID 的需求已经增加。多数 Re-ID 方法采用深度度量学习来映射嵌入向量，这些嵌入向量对应的正样本在向量空间中彼此距离较近，而负样本彼此距离较远。

与自监督方法通过各种数据增强方法合成正样本不同，在 Re-ID 中，正样本包含具有相同 ID 但属于不同场景的图像。换句话说，摄像机拍摄的背景和障碍物等特征会有所不同，但如果身份相同，嵌入矢量之间的距离应该在矢量空间较小。由于背景和角度随场景的变化而变化，从网络中提取的特征不可避免地不仅包括与身份相关的信息，还包括与摄像机域/身份无关的信息。这个问题与视频 Re-ID 更相关，因为它使用几个连续的帧作为输入。

与图像 Re-ID相比，视频 Re-ID 提供了时间信息，减少了外观的不确定性。然而，随着与 ID 无关的语义也增加，网络可能容易受到类内噪声的影响。大多数方法都利用时间信息，例如在帧之间获取，根据先前帧特征选择性地捕获细节，以及提取过去序列中不存在的特征。然而，这些方法在很大程度上取决于网络在多帧期间提取目标正确特征的能力。

例如，由于帧之间的时间间隔很小，相似的背景会重复出现，因此网络经常将背景学习为关键特征，如图 1-(a) 中的激活图所示，例如虚线框区域。如图 1-(a) 所示，最终的嵌入包含了砖块和出租车的语义，因此检索到与查询具有相似外观和背景的案例。事实上，不必要的背景信息在序列中重复并包含在最终特征中，这导致网络确定出现在与查询具有相似外观和背景的场景中的人的向量比查询更接近查询同一个人出现在具有不同外观和背景的场景中的向量（场景占比会变大，更容易受到场景的影响）。最终，如果网络迫使具有此类纠缠信息的最终嵌入向量更接近向量空间，则特征的表示能力会降低。

在本文中，我们提出了一种学习方法，可以解开特征并利用它们来有效地利用网络的能力，而不损害表征能力。如图 1-(b,c) 所示，当特征分离时，网络完全专注于每个任务，一致的丰富信息起到枢纽作用，找到正确的片段作为排名 1。我们提出的方法，解开损失和相机 ID 分类，强制将关注身份的特征与具有重复背景和遮挡模式的特征分开，具体取决于所选相机。此外，目标定位模块 (TLM) 使与 ID 相关的嵌入对于边界框未对准（视频任务中长期存在的问题）更加稳健。 通过交换和聚合来利用分离的特征，网络不仅可以学习显式分离特征，还可以隐式地提高能力。我们提出的方法表明，根据视频 Re-ID 中身份信息的存在或不存在，它对于分离非常有用。

总而言之，我们的主要贡献有三方面：

  • 我们提出了一种新颖的 DSANet，它将特征分解为 ID 代表特征和相机特征特征，并为辅助任务创建这两者的新组合。
  • 我们引入了 ID 表示学习，这使得最终的嵌入特征对于边界框未对齐更加鲁棒，并且能够区分时间流。
  • 广泛的实验不仅定量地证明了我们的 DSANet 具有优异的性能和具有竞争力的参数量，而且定性地证明了解纠缠学习的性能与我们预期的一样。

Person re-identification(Re-ID) aims at matching the identity of a specific person on camera from various positions and angles. The demand for Re-ID has risen owing to the proliferation of intelligent surveillance systems and various multimedia applications. Most Re-ID approaches employ deep metric learning to map embedding vectors corresponding to positive samples being closer to each other in vector space, and negative samples being farther away from each other.

Unlike the self-supervised approaches that compose positive samples through various data augmentation approaches, in Re-ID, positive samples comprise images having the same ID but belonging to different scenes. In other words, there are differences in characteristics such as background and obstacles depending on the camera taken, but if the ID is the same, the distance between embedding vectors should be low in vector space. Features extracted from the network inevitably include not only ID-related information but also camera domain/ID-unrelated information because the background and angle change depending on the scenes. This issue is more pertinent to video Re-ID, which uses several consecutive frames as input.

Compared to image Re-ID, video Re-ID provides temporal information which reduces the uncertainty of appearance. However, as ID-unrelated semantics also increase, the network may be vulnerable to intra-class noise. Most methodologies exploit temporal information such as obtaining between frames, selectively capturing details based on the previous frame features, and extracting features that were not in the past sequence. However, these methods depend heavily on the network’s capability extracting the correct features for the target during multiple frames.

For instance, as the time interval between frames is small, a similar background is repeated, so the network frequently learns the background as a key feature as in the activation map in Figure 1-(a), e.g. the dashed-box areas. As shown in Figure 1-(a), the final embedding includes the semantics of bricks and taxis, so cases with a similar appearance and background to the query are retrieved. In actual fact, unnecessary background information is repeated in the sequence and included in the final feature, which leads the network to determine that a vector of a person appearing in a scene having a similar appearance and background as the query is closer to the query than vectors of the same person appearing in a scene having a different appearance and background. Ultimately, if the network is forcing the final embedding vector with such entangled information to be closer to the vector space, the representational ability of the feature reduces.

In this paper, we propose a learning method that disentangles features and utilizes them to efficiently harness the network’s capability without harming the representational ability. As briefly illustrated in Figure 1-(b, c), when the features are separated, the network fully focuses on each task, and the consistent rich information takes a pivot role to find the correct snippet as rank-1. Our proposed method, disentangling loss and camera ID classification, forces to separate features focused on identity from features having repeated background and occlusion patterns depending on the camera chosen. Furthermore, the target localization module (TLM) makes ID-related embeddings more robust against bounding box misalignment, a chronic concern in video tasks. By leveraging the disentangled features through switching and aggregation, the network learns to not only explicitly（显式的） separate features but also implicitly increase capability. Our proposed method shows that it can be quite useful for separation depending on the presence or absence of identity information in video Re-ID.

In summary, our main contributions are three-fold:

  • We propose a novel DSANet that disentangles features into ID representative features and camera characteristic features and creates a new combination of these two for auxiliary tasks.
  • We introduce ID representation learning, which makes the final embedding features more robust to bounding box misalignment and discriminative to temporal flow.
  • Extensive experiments not only quantitatively demonstrate the excellent performance and competitive amount of parameters of our DSANet but also qualitatively prove disentanglement learning performs as we intended.

2. Related Works

2.1 [[Video-based Person Re-Identification]]

在视频Re-ID中，完全提取目标特征、排除干扰因素至关重要。大多数方法通过利用给定的信息（空间和时间）来获得鲁棒的视频级表示。 3D CNN 能够对局部时间关系和相对位置进行编码，与非局部操作或注意机制结合使用，该注意机制对齐每个部分以防止最终嵌入的恶化。一些研究充分利用全局-局部时间线索来预测每个帧的权重。陈等人关注的是对象，一个时不变的特征，而不是提取运动向量。它将特征分解为时间相干性和时间运动，并通过在多个尺度上采样噪声来提取判别特征。然而，区分性特征不一定是一成不变的。例如，根据视频序列，背景和障碍物可能不会随时间维度变化；相反，与 ID 相关的语义可能会因人体姿势而有所不同。

最流行的方法 [6,13,15,29,33] 利用丰富的时空线索。侯等人 [13]通过自适应选择时间核尺度来考虑时间关系，然后提取对于多尺度而言鲁棒的空间特征。为了使最终的表示能够抵抗空间和时间的干扰，Eom 等人[6]识别干扰模式，将其存储在内存中，并通过障碍物处理来细化视频级特征。这些方法引导网络提取目标人物的丰富表示，但需要复杂的网络结构和繁重的计算。根据[6]观察到模式出现在背景和障碍物中，我们的方法明确地解开了 ID 和背景的特征。与[1]通过计算均值获得时间相干性特征不同，我们提出了一个辅助任务，可以补充解缠结过程并通过特征增强增强网络的辨别力。

In video Re-ID, it is crucial to completely extract the features of the target, excluding distractors. Most methods obtain robust video-level representations by utilizing the given information: spatial and temporal. A 3D CNN, capable of encoding the local temporal relation and the relative position, is used in conjunction with a non-local operation [8] or an attention mechanism [17] that aligns each part to prevent deterioration of the final embeddings. Some studies [7, 21] fully harness global-local temporal clues to predict the weights of each frame. Chen et al. [1] focuses on the object, a time-invariant feature, rather than on extracting the motion vector. It disentangles features into temporal coherence and temporal motion and distills the discriminative characteristic by sampling noise at multiple scales. However, discriminative features are not necessarily immutable. For instance, depending on the video sequences, the background and obstacle may not change with a time dimension; rather, ID-related semantics may be variant because of the human pose.

The most popular approach [6, 13, 15, 29, 33] takes advantage of rich spatio-temporal clues. Hou et al. [13] considers temporal relations by adaptively selecting the temporal kernel scale and then extracts spatial features that are robust for multi-scaling. To make the final representation resistant to spatial and temporal distractors, Eom et al. [6] identifies the distractor pattern, stores it in memory, and refines the video-level features through obstacle handling. These methods guide the network to extract rich representations of the target person but require a complex network structure and heavy computation. In light of [6]’s observation that patterns occur in backgrounds and obstacles, our method explicitly disentangles the features of the ID and background. Unlike [1], which obtains temporal coherence features by computing the mean, we propose an auxiliary task that can complement the disentangling process and enhance the discrimination of the network through feature augmentation.

2.2 Feature Disentanglement

特征解缠结方法分解representations，使它们具有独立且有意的属性，然后进一步利用它们。它在各个领域都有不同的用途[4,16,19]，主要是在域适应领域[20,40]。郑等人 [36]在Re-ID领域应用特征解缠结。它有两个不同的编码器，可以根据外观和结构代码来分离特征。该网络通过切换代码生成高质量的crossID合成图像，并利用生成的图像进行在线学习。这一系列的过程是端到端进行的。该方法成功地重建了以混合外观和结构为特征的图像，但未能考虑外观代码中与 ID 相关的精细细节对应的配饰和头发。正如[39]中提到的，赋予判别力的细节是关键因素，但它仍然保留在结构代码中，可能会混淆网络学习。邹等人[40]将嵌入分解为与 ID 相关/不相关的特征，并仅使用与 ID 相关的特征来减少巨大的领域差距。它具有编码器-解码器结构，利用循环一致性特征重建原始图像，将特征分离为外观和结构因素。

与上述方法不同，我们的方法可以令人满意地解开特征，而不需要另一个网络。此外，解缠结向量的切换和聚合发生在特征级别，无需重建图像。

The feature disentangling method separates representations so that they have independent and intentional attributes and then utilizes them further. It is applied for diverse purposes [4, 16, 19] in various areas, mainly in the domain adaptation field [20, 40]. Zheng et al. [36] applies feature disentanglement in the Re-ID field. It has two distinct encoders to separate features based on appearance and structure code. The network generates high-quality crossID composed images by switching the codes and performs online learning with the generated images. This series of processes goes on end-to-end. This method succeeds in reconstructing images characterized by mixed appearance and structure but fails to consider accessories and hair corresponding to ID-related fine details in the appearance code. As mentioned in [39], the fine detail that imparts discriminant power is the crucial factor, but it remains in the structure code and may confuse network learning. Zou et al. [40] disentangles embeddings into ID-related/unrelated features and uses only ID-related features to reduce the huge domain gap. It has an encoder–decoder structure that reconstructs the original image using the cycle consistency characteristic to separate features into appearance and structure factors.

Unlike the above methods, our method can satisfactorily disentangle features without requiring another network. In addition, switching and aggregation with disentangled vectors occurs at the feature level with no need to reconstruct the image.

3. Proposed Method

3.1 DSANet

图 2 简要概述了我们的解缠结、交换和聚合网络 (DSANet)。DSANet 涉及三个过程：特征解缠结、ID 表示学习和辅助任务。 DSANet 明确地将features分为两类：ID features和相机features。然后，通过目标定位模块（TLM）和帧权重生成（FWG） 对提取的ID特征进行ID表示学习，使其更能抵抗目标位置的变化。最后，DSANet 通过利用交换和聚合操作（SAO） 来执行辅助任务，该操作增强了具有各种摄像机特性的功能，以提高网络的识别能力。这一系列过程端到端学习，每个组件单独或协作工作，以实现 Re-ID 的首要目标：鲁棒且差异化的特征提取。以下各节对每个部分进行详细说明。

A brief overview of our Disentanglement and Switching and Aggregation Network (DSANet) is given in Figure 2. DSANet involves three processes: feature disentangling, ID representation learning, and auxiliary tasks. DSANet explicitly separates features into two categories: ID features and camera features. Then, ID representation learning is performed for the extracted ID feature to become more resistant to changes in the target’s position through Target Localization Module (TLM) and Frame Weight Generation (FWG). Finally, DSANet conducts auxiliary tasks by utilizing Switching and Aggregating Operation (SAO), which augments features to have various camera characteristics to improve the network’s discernment. The series of processes learn end-to-end, and each component works individually and cooperatively to achieve the foremost goal of Re-ID: robust and differential feature extraction. The following sections present a detailed description of each part.

3.2 Feature Disentangling

通道扩展层 (CEL) 现有方法 [19、36、40] 需要多个网络来解开特征。
然而，DSANet 通过划分backbone的分支，令人满意地将 FIDF_{ID}FID​ 和 FcamF_{cam}Fcam​ 从一个网络中分离出来。

Channel Expansion Layer (CEL) Existing methods [19, 36, 40] require multiple networks to disentangle features. However, DSANet separates FIDF_{ID}FID​ and FcamF_{cam}Fcam​ from one network satisfactorily by dividing the branches of the backbone. The starting point of dividing branches is the last layer of ResNet-50 [9] before extracting more discriminative features. For FcamF_{cam}Fcam​ to separate reliably, we need the disentangling loss LdisL_{dis}Ldis​ and the camera ID classification.

Disentangling Loss Considering that fID∈Rc×1×1\textbf{f}_{ID}\in \mathbb{R}^{c\times 1\times 1}fID​∈Rc×1×1 and fcam∈Rc×1×1\textbf{f}_{cam}\in \mathbb{R}^{c\times 1\times 1}fcam​∈Rc×1×1 should have different information, we measure the cosine similarity of these two vectors as follows:

Ldis=max⁡(fID⋅fcam∣∣fID∣∣2∣∣fcam∣∣2,0)L_{dis} = \max(\frac{\textbf{f}_{ID}\cdot\textbf{f}_{cam}}{||\textbf{f}_{ID}||_2||\textbf{f}_{cam}||_2},0) Ldis​=max(∣∣fID​∣∣2​∣∣fcam​∣∣2​fID​⋅fcam​​,0)

where fcam\textbf{f}_{cam}fcam​ is vector maxpooling in the feature maps Fcam\textbf{F}_{cam}Fcam​. When the cosine distance of fID\textbf{f}_{ID}fID​ and fcam\textbf{f}_{cam}fcam​ becomes 0, it indicates dissimilarity between the two vectors. A margin was given to the loss function because being exactly opposite could harm the representation ability. Disentangling loss allows DSANet to extract fID\textbf{f}_{ID}fID​ and fcam\textbf{f}_{cam}fcam​ focused on each task without being disturbed by spatial distractors in a situation where consecutive frames are input.

3.3 ID representation Learning

Target Localization Module (TLM)

目标定位模块 (TLM) 为了即使在连续帧中也能从网络中一致地提取 FID\textbf{F}_{ID}FID​，它必须对边界框未对齐具有鲁棒性。 Re-ID 的挑战之一是，由于检测器性能的限制，目标在裁剪后的视频剪辑中会倾斜。因此，在视频Re-ID中，representation必须根据移动的感兴趣的人的位置遵循相应的信息。为了实现上述目标，受 SAM [30] 的启发，我们提出了一种进行空间注意力的 TLM。 它旨在关注显着信息在一个完整特征图中的位置。然而，我们提出的 TLM 将特征图 FID\textbf{F}_{ID}FID​ 在空间维度分为 ZL\textbf{Z}_LZL​、ZM\textbf{Z}_MZM​ 和 ZR∈Rc×t×h×ω2\textbf{Z}_R\in\mathbb{R}^{c\times t\times h\times\frac{\omega}{2}}ZR​∈Rc×t×h×2ω​如图3所示，并关注目标在每个区域的位置。我们串联划分后的特征图Z\textbf{Z}Z的通道维度上最大池化和平均池化的结果。然后我们通过卷积层和softmax获得空间注意力图A。

Target Localization Module (TLM) To consistently extract FID\textbf{F}_{ID}FID​ from the network even in consecutive frames, it must be robust to bounding box misalignment. One of the challenges in Re-ID is that the target is skewed in cropped video clips owing to the limits of the detector’s performance. Therefore, in video Re-ID, the representation must follow the corresponding information according to the position of the moving person of interest.To achieve the above goal, we propose a TLM that conducts spatial attention, inspired by SAM [30]. It is designed to focus on where salient information is located in one complete feature map. However, our proposed TLM divides the feature map FID\textbf{F}_{ID}FID​ into ZL\textbf{Z}_LZL​, ZM\textbf{Z}_MZM​ , and ZR∈Rc×t×h×ω2\textbf{Z}_R\in\mathbb{R}^{c\times t\times h\times\frac{\omega}{2}}ZR​∈Rc×t×h×2ω​ in the spatial dimension as shown in Figure 3 and pays attention to the location of the target in each region.We concatenate the result of max pooling and average pooling in the channel dimension of the divided feature maps Z\textbf{Z}Z. A=Softmax(Conv([Max(Z);Avg(Z)]))A = Softmax(Conv([Max(\textbf{Z}); Avg(\textbf{Z})])) A=Softmax(Conv([Max(Z);Avg(Z)]))

In many cases empirically, given that the person is located in the center [3, 18], we add information corresponding to the middle AMA_MAM​ to make the final attention map ZattnZ_{attn}Zattn​.

Zattn=[AL;AR]+AMZ_{attn} = [A_L;A_R] + A_M Zattn​=[AL​;AR​]+AM​

Ft=Zattn⊙FID+FIDF_t = Z_{attn}\odot\textbf{F}_{ID}+\textbf{F}_{ID} Ft​=Zattn​⊙FID​+FID​

LL/R=CE(P(ft/L))+CE(P(P(ft/R))L _ { L / R } = C E ( P ( \textbf{f} _ { t / L } ) ) + C E ( P ( P ( \textbf{f} _ { t / R } ) ) LL/R​=CE(P(ft/L​))+CE(P(P(ft/R​))

where CECECE stands for cross-entropy loss, PPP indicates the linear classifier computing probabilities, and ft/L∈Rc×t×1×1\textbf{f}_{t/L}\in\mathbb{R}^{c\times t\times 1\times 1}ft/L​∈Rc×t×1×1 and ft/R∈Rc×t×1×1\textbf{f}_{t/R}\in\mathbb{R}^{c\times t\times 1\times 1}ft/R​∈Rc×t×1×1 are vectors maxpooling in the feature maps corresponding to the left and right sides of Ft\textbf{F}_tFt​.

Frame Weight Generation (FWG)

Frame Weight Generation (FWG) 大多数方法[26, 38]主要使用全局平均池化将视频序列的表示合并到最终的嵌入向量中。为了进一步利用时间信息，我们通过 FWG 预测每帧的权重 w^∈R1×t\hat{\textbf{w}}\in\mathbb{R}^{1\times t}w^∈R1×t 并反映它以获得最终的嵌入向量。 为了让 FWG 做出合理的预测，我们根据表示帧重要性指数的值生成一个伪标签，并将其用于训练。ft∈Rc×t×1×1\textbf{f}_{t}\in\mathbb{R}^{c\times t\times 1\times 1}ft​∈Rc×t×1×1是通过在空间维度上从TLM中提取的Ft∈Rc×t×h×w\textbf{F}_{t}\in\mathbb{R}^{c\times t\times h\times w}Ft​∈Rc×t×h×w进行最大池化得到的。ft\textbf{f}_tft​ 包含每个帧的紧凑特征，该特征被输入到 FWG 并用于生成伪标签。我们计算分类任务中的交叉熵，将其定义为损失函数，并学习减少该值。换句话说，交叉熵值越小，该任务的特征越正确。我们可以利用这个事实来确定哪一帧包含更多的身份信息，并将这个值表示为概率值。 在按照帧分解ft\textbf{f}_tft​后，分别计算与ID标签的交叉熵。然后，在获得的值中找到最大值，并根据这些值计算差值ddd。我们将此计算表示为图 4 中的逆运算。

d=max⁡(CE(P(ft)))−CE(P(ft))d = \max(CE(P(\textbf{f}_t)))-CE(P(\textbf{f}_t)) d=max(CE(P(ft​)))−CE(P(ft​))

我们让这些值的概率对应于帧的重要性，然后将其用作伪标签w\textbf{w}w。 FWG 由一个卷积层组成，将通道维度减少到 1。通过总结帧特征来预测权重。最终权重w^\hat{\textbf{w}}w^通过与伪标签生成相同的操作过程进行预测。我们通过对 ft\textbf{f}_tft​ 上的 w^\hat{\textbf{w}}w^ 进行时间关注 (temperal attention) 来获得最终表示 fID\textbf{f}_{ID}fID​。

Frame Weight Generation (FWG) Most methods [26, 38] mainly use global average pooling to merge the representation of the video sequence into the final embedding vector.To further utilize temporal information, we predict the weight w^∈R1×t\hat{\textbf{w}}\in\mathbb{R}^{1\times t}w^∈R1×t for each frame through FWG and reflect this to obtain the final embedding vector. For the FWG to make reasonable predictions, we generate a pseudo label from the value that represents the importance index of the frame and use it for training.ft∈Rc×t×1×1\textbf{f}_{t}\in\mathbb{R}^{c\times t\times 1\times 1}ft​∈Rc×t×1×1 was obtained by max-pooling of Ft∈Rc×t×h×w\textbf{F}_{t}\in\mathbb{R}^{c\times t\times h\times w}Ft​∈Rc×t×h×w extracted from TLM in the spatial dimension. ft\textbf{f}_tft​ contains a compact feature for each frame, which is input to the FWG and also used to generate a pseudo label. We compute the cross-entropy in the classification task, define it as a loss function, and learn to reduce this value. In other words, the smaller the crossentropy value, the more correct the feature for the task. We can use this fact to determine which frame contains more identity information and express this value as a probability value. After dividing ft\textbf{f}_tft​ by frame, the cross-entropy with the ID label was computed respectively. Then, the max value was found among the obtained values, and the difference ddd was calculated based on these values. We express this computation as a reverse operation in Figure 4.

d=max⁡(CE(P(ft)))−CE(P(ft))d = \max(CE(P(\textbf{f}_t)))-CE(P(\textbf{f}_t)) d=max(CE(P(ft​)))−CE(P(ft​))

We make these values’ probabilities correspond to the importance of the frame and then use it as a pseudo-label w\textbf{w}w. FWG consists of one convolution layer reducing the channel dimension to 1. The weight is predicted by summarizing the frame features. The final weight w^\hat{\textbf{w}}w^ is predicted through the same operation process as pseudo label generation. We obtain the final representation fID\textbf{f}_{ID}fID​ by taking temporal attention to w^\hat{\textbf{w}}w^ on ft\textbf{f}_tft​.

Frame Weight Loss

我们计算伪标签 w\textbf{w}w 和预测 w^\hat{\textbf{w}}w^ 之间的均方误差，并将其用作损失函数 LwL_wLw​ 以期望来自 FWG 的可靠权重。我们不使用KL散度来减少w\textbf{w}w和w^\hat{\textbf{w}}w^之间的差异，因为概率值越接近0，就越难衡量相似度分布之间。我们还通过实验确定均方误差更适合w^\hat{\textbf{w}}w^的合理预测。

We calculate the mean squared error between pseudo label w\textbf{w}w and predicted w^\hat{\textbf{w}}w^ and use it as a loss function LwL_wLw​ to expect reliable weights from the FWG. We do not use KL-divergence to reduce the difference between w\textbf{w}w and w^\hat{\textbf{w}}w^ because the closer the probability value is to 0, the more difficult it is to measure the similarity between distributions. We also experimentally determine that the mean squared error is more appropriate for the reasonable prediction of w^\hat{\textbf{w}}w^.

Intra-class Loss

为了减少小批量正样本之间的类内噪声，我们将其定义如下：

Lic=∑B(∑fID∈Bk1Bk(fID−1Bk∑fID∈BkfID)2),L_{ic} = \sum^B(\sum_{f_{ID}\in B_k}\frac{1}{B_k}(\textbf f_{ID}-\frac{1}{B_k}\sum_{f_{ID}\in B_k}\textbf f_{ID})^2), Lic​=∑B​(fID​∈Bk​∑​Bk​1​(fID​−Bk​1​fID​∈Bk​∑​fID​)2),

其中 BkB_kBk​ 表示小批量中属于身份 kkk 的特征集。属于正样本的fID\textbf{f}_{ID}fID​是从各个序列中提取出来的，但是场景对应的语义已经被去掉了，所以只能得到ID相关的信息。类内损失函数使得fID\textbf{f}_{ID}fID​更加关注正样本中常见但独特的特征，减轻了训练的负担。

这个损失函数的意思相当于就是让所有的ID特征尽可能的相似，向平均值靠拢

To reduce intra-class noise among the positive samples of the mini-batch, we define it as follows:

Lic=∑B(∑fID∈Bk1Bk(fID−1Bk∑fID∈BkfID)2),L_{ic} = \sum^B(\sum_{f_{ID}\in B_k}\frac{1}{B_k}(\textbf f_{ID}-\frac{1}{B_k}\sum_{f_{ID}\in B_k}\textbf f_{ID})^2), Lic​=∑B​(fID​∈Bk​∑​Bk​1​(fID​−Bk​1​fID​∈Bk​∑​fID​)2),

where BkB_kBk​ denotes the feature set belonging to identity kkk in the mini-batch. The fID\textbf{f}_{ID}fID​ belonging to the positive samples is extracted from various sequences, but the semantic corresponding to the scene has been removed, so only the IDrelated information is available. The intra-class loss function allows fID\textbf{f}_{ID}fID​ to focus more on common but distinctive features within positive samples and reduces the burden on training.

3.4 Auxiliary tasks（辅助任务）

Camera ID Classification

虽然 FID\textbf{F}_{ID}FID​ 和 Fcam\textbf{F}_{cam}Fcam​ 的差异性是通过 CEL 和解缠损失来保证的，但fcam\textbf{f}_{cam}fcam​需要一个指导来根据相机自适应地掌握背景和遮挡模式。图 5 说明了场景和障碍物模式的特征是相似的，具体取决于为每个数据集拍摄的相机。考虑到上述事实，DSANet 使用包含相机特征的 fcam\textbf{f}_{cam}fcam​ 执行预测相机 ID 的辅助任务。由于捕获序列的摄像机 ID 已被标记，因此不需要额外的注释。相反，在积极利用给定标签的同时，辅助任务引导 fcam\textbf{f}_{cam}fcam​ 更多地关注背景，排除目标。用于相机ID分类的损失函数是交叉熵损失，计算如下：

Lcam=−ccamilog⁡(ewifcami∑j=1Ccamewjfcamj)L_{cam} = -c^i_{cam}\log(\frac{e^{\textbf w_i\textbf f^i_{cam}}}{\sum_{j=1}^{C_{cam}}e^{\textbf w_j\textbf f^j_{cam}}}) Lcam​=−ccami​log(∑j=1Ccam​​ewj​fcamj​ewi​fcami​​)

其中W\textbf{W}W是全连接层的权重矩阵，ccamic^i_{cam}ccami​是相机ID标签，CcamC_{cam}Ccam​是相机ID总数。

Although the dissimilarity of FID\textbf{F}_{ID}FID​ and Fcam\textbf{F}_{cam}Fcam​ is guaranteed through CEL and disentangling loss, a guide is needed for fcam\textbf{f}_{cam}fcam​ to adaptively grasp the background and occlusion patterns depending on the camera. Figure 5 illustrates that the characteristics of the scene and obstacle patterns are similar depending on the camera taken for each dataset. Taking the above facts into account, DSANet performs the auxiliary task of predicting the camera ID with fcam\textbf{f}_{cam}fcam​ containing the camera characteristics. As the camera IDs that captured the sequence are labeled, no additional annotation is needed. Rather, while actively utilizing the given label, the auxiliary task guides fcam\textbf{f}_{cam}fcam​ to focus more on the background, excluding the target. The loss function used for camera ID classification is the cross-entropy loss calculated as follows:

Lcam=−ccamilog⁡(ewifcami∑j=1Ccamewjfcamj)L_{cam} = -c^i_{cam}\log(\frac{e^{\textbf w_i\textbf f^i_{cam}}}{\sum_{j=1}^{C_{cam}}e^{\textbf w_j\textbf f^j_{cam}}}) Lcam​=−ccami​log(∑j=1Ccam​​ewj​fcamj​ewi​fcami​​)

where W\textbf{W}W is the weight matrix of the fully connected layer, ccamic^i_{cam}ccami​ is the camera ID label, and CcamC_{cam}Ccam​ is the total number of camera IDs.

Switching and Aggregating Operation (SAO)

解开的 fID\textbf{f}_{ID}fID​ 和 fcam\textbf{f}_{cam}fcam​ 可以通过切换和聚合(SAO)使用新的嵌入向量进行增强。具体来说，当构成一个小批量的fID\textbf{f}_{ID}fID​和fcam\textbf{f}_{cam}fcam​被随机交换和重新组合时，各种fID\textbf{f}_{ID}fID​ 和 fcam\textbf{f}_{cam}fcam​的pairs就已生成。即使分配了新的fcam\textbf{f}_{cam}fcam​，fID\textbf{f}_{ID}fID​的标签对应的ID也不会改变。因此，通过SAO得到的嵌入向量faug\textbf{f}_{aug}faug​的ID是由fID\textbf{f}_{ID}fID​决定的，所以我们使用faug\textbf{f}_{aug}faug​ 用于预测 ID 的另一个辅助任务。该任务不仅提高了网络对各种相机场景的鲁棒性，而且还增强了自适应解缠结学习。

有点像把人物放入不同的相机中进行数据增强

The disentangled fID\textbf{f}_{ID}fID​ and fcam\textbf{f}_{cam}fcam​ can be augmented using a new embedding vector through switching and aggregation. Specifically, when fID\textbf{f}_{ID}fID​ and fcam\textbf{f}_{cam}fcam​ constituting a mini-batch are randomly swapped and recombined, various pairs of fID\textbf{f}_{ID}fID​ and fcam\textbf{f}_{cam}fcam​ are generated. Even if a new fcam\textbf{f}_{cam}fcam​ is assigned, the ID corresponding to the label of fID\textbf{f}_{ID}fID​ does not change. Therefore, the ID of the embedding vector faug\textbf{f}_{aug}faug​ obtained through SAO is determined by fID\textbf{f}_{ID}fID​, so we use faug\textbf{f}_{aug}faug​ for another auxiliary task of predicting the ID. This task not only improves the robustness of the network to various camera scenes but also enhances the disentangling learning adaptively.

3.5 Training and Testing Phases

在训练阶段，我们采用通过fID\textbf{f}_{ID}fID​计算的三元组损失[12]和上一节中描述的损失。 DSANet 的总体目标函数定义为：

Ltotal=Lce+Ltri+Ldis+Lic+λLwL_{total} = L_{ce}+L_{tri}+L_{dis}+L_{ic}+\lambda L_{w} Ltotal​=Lce​+Ltri​+Ldis​+Lic​+λLw​

其中 λ\lambdaλ 是比例因子并设置为 0.1。

在测试阶段，我们仅使用 fID\textbf{f}_{ID}fID​ 作为最终表示，并使用余弦相似度计算最终嵌入向量之间的距离。

In the training phase, we employ the triplet loss [12] calculated with fID\textbf{f}_{ID}fID​ and losses described in the previous section. The overall objective function of DSANet is defined as:

Ltotal=Lce+Ltri+Ldis+Lic+λLwL_{total} = L_{ce}+L_{tri}+L_{dis}+L_{ic}+\lambda L_{w} Ltotal​=Lce​+Ltri​+Ldis​+Lic​+λLw​

where λ\lambdaλ is the scale factor and is set to 0.1.

In the testing phase, we use fID\textbf{f}_{ID}fID​ only as the final representation and calculate the distance between the final embedding vectors using cosine similarity.
