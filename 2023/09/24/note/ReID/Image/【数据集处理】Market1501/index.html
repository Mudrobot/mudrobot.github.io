【数据集处理】Market1501

数据集简介

Market-1501 is a large-scale public benchmark dataset for person re-identification. It contains 1501 identities which are captured by six different cameras, and 32,668 pedestrian image bounding-boxes obtained using the Deformable Part Models pedestrian detector. Each person has 3.6 images on average at each viewpoint. The dataset is split into two parts: 750 identities are utilized for training and the remaining 751 identities are used for testing. In the official testing protocol 3,368 query images are selected as probe set to find the correct match across 19,732 reference gallery images.

Market-1501是一个用于人员重新识别的大型公共基准数据集。它包含由六个不同摄像机捕获的 1501 个身份，以及使用Deformable Part Models pedestrian detector获得的32668个行人图像边界框。每个人在每个视点平均有3.6张图像。数据集分为两部分：750 个身份用于训练，其余 751 个身份用于测试。在官方测试协议中，选择 3,368 个查询图像作为query集，以在 19,732 个参考库图像中找到正确的匹配项（gallery集）。

数据集结构

  • bounding_box_test：这个文件夹中包含19732张图像用于测试 (gallery集)
  • bounding_box_train：这个文件夹中包含12936张图片用于训练 (train集)
  • query：文件夹中包含3368张查询图片 (query集)
  • gt_bbox：It contains 25259 images, all hand-drawn. The images correspond to all the 1501 individuals in the test and training set.It is used to distinguish “good” “junk” and “distractors”.
  • gt_query：对于3368个查询中的每一个，都有好的和垃圾相关的图像(包含同一个人)。此文件包含良好和垃圾图像的图像索引。它用于性能评估。（.mat文件）

我们在这次训练中使用的文件夹主要是bounding_box_train,query以及bounding_box_test。分别对应训练集，query集和gallery集
