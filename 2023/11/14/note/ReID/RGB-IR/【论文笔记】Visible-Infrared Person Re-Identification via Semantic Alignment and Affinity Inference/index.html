【论文笔记】Visible-Infrared Person Re-Identification via Semantic Alignment and Affinity Inference

  • Authors: [[Xingye Fang]], [[Yang Yang]], [[Ying Fu]]
  • PDF Attachments
    - Fang et al. - Visible-Infrared Person Re-Identification via Sema.pdf

文章内容

Abstract

Visible-infrared person re-identification (VI-ReID) focuses on matching the pedestrian images of the same identity captured by different modality cameras. The part-based methods achieve great success by extracting fine-grained features from feature maps. But most existing part-based methods employ horizontal division to obtain part features suffering from misalignment caused by irregular pedestrian movements. Moreover, most current methods use Euclidean or cosine distance of the output features to measure the similarity without considering the pedestrian relationships. Misaligned part features and naive inference methods both limit the performance of existing works. We propose a Semantic Alignment and Affinity Inference framework (SAAI), which aims to align latent semantic part features with the learnable prototypes and improve inference with affinity information. Specifically, we first propose semantic-aligned feature learning that employs the similarity between pixelwise features and learnable prototypes to aggregate the latent semantic part features. Then, we devise an affinity inference module to optimize the inference with pedestrian relationships. Comprehensive experimental results conducted on the SYSU-MM01 and RegDB datasets demonstrate the favorable performance of our SAAI framework. Our code will be released at https://github.com/xiaoye-hhh/SAAI.

可见光-红外行人重识别（VI-ReID）专注于匹配不同模态摄像头拍摄的同一身份的行人图像。基于部分的方法通过从特征图中提取细粒度特征取得了巨大成功。但现有的大多数基于部位的方法采用水平分割来获取因行人不规则运动而导致错位的部位特征。此外，当前大多数方法使用输出特征的欧几里德距离或余弦距离来测量相似度，而不考虑行人关系。未对齐的零件特征和朴素的推理方法都限制了现有作品的性能。我们提出了语义对齐和亲和力推理框架（SAAI），旨在将潜在语义部分特征与可学习原型对齐，并利用亲和力信息改进推理。具体来说，我们首先提出语义对齐特征学习，利用像素特征和可学习原型之间的相似性来聚合潜在语义部分特征。然后，我们设计了一个亲和力推理模块来优化行人关系的推理。在 SYSU-MM01 和 RegDB 数据集上进行的综合实验结果证明了我们的 SAAI 框架的良好性能。我们的代码将发布在 https://github.com/xiaoye-hhh/SAAI 。

总结

之前工作存在的问题

图1：**框架差异** (a) 大多数现有的基于部分的方法从特征图中提取水平部分特征，并采用朴素推理（例如欧几里德距离和余弦距离）来匹配图像。不同的是，我们的方法提取潜在语义部分特征并利用相似度信息。 (b)我们的方法提取的零件特征具有更好的语义信息。 (c) 我们的亲和力推断利用行人关系来优化距离计算。线条代表图像关系。线越粗表示亲和力越高。绿色和红色圆圈分别表示正面和负面图库图像。

VI-ReID 受到显着的模态差异和其他因素（例如观点、背景和运动）的影响。导致类内差异较大的因素使得匹配变得困难。图 1 (a) 显示现有方法 [13,23,24,35] 提取水平部分特征来缓解这个问题。图 1 (b) 说明行人部位（例如手臂）由于运动而没有位于固定位置。因此，简单的固定高度的水平划分会导致部分特征变成语义错位，从而限制性能。

此外，图 1 (a) 显示大多数现有方法采用朴素推理方法来计算查询图像和图库图像之间的相似度。如图 1 © 所示，这些方法通过将图库图像视为单个实体来忽略它们之间的亲和力信息。图库图像属于相同模态，没有模态差异，这可以提供有用的亲和力信息来改进匹配。因此，SIM [14]建议利用图库图像之间的相似性并计算 Jaccard 距离来增强匹配。然而，在计算杰卡德距离时，元素关系是二元关系，没有充分利用亲和度信息。

主要贡献

我们提出了Semantic Alignment and Affinity Inference framework (SAAI)，它对齐潜在语义部分特征并更好地利用辅助亲和力信息。 SAAI 框架由语义对齐特征学习（SAFL）和相似性推理模块（AIM）组成。

  • 我们提出了一种用于 VI-ReID 的端到端语义对齐和相似度推理框架（SAAI），以探索语义对齐特征学习和相似度推理方法的联合应用
  • 我们提出了一种语义对齐特征学习（SAFL）来对齐潜在语义部分特征。此外，我们设计了一个相似性推理模块（AIM）来利用行人关系进行匹配。
  • 综合实验结果证明了所提出框架的有效性。与各种测试设置中最先进的方法相比，它实现了卓越的性能。

方法

在本节中，我们首先定义问题并提出总体框架。随后，我们详细阐述了语义对齐特征学习（SAFL）和亲和推理模块（AIM）的设计细节。最后，我们提供了 SAAI 的总体损失。

3.1 Formulation and Overview

问题定义

我们使用Q={qi∣i=1,2,⋯ ,Nq}Q=\{q_i|i=1,2,\cdots,N_q\}Q={qi​∣i=1,2,⋯,Nq​}和G={gj∣j=1,2,⋯ ,Ng}G=\{g_j|j=1,2,\cdots,N_g\}G={gj​∣j=1,2,⋯,Ng​}来代表有NqN_qNq​个图像的query set以及NgN_gNg​个图像的gallery set。query set和gallery set中的图像属于不同的模态。VI-ReID的目的就是为了建立query set和gallery set之间的联系。

Overview

SAAI结构图

我们的模型主要包含两个部分：

  1. 语义对齐特征学习旨在通过像素级特征和可学习原型之间的相似性来提取潜在语义部分特征。直觉是，有意义的部分应该由具有相似内容的特征组成，从而形成部分一致性。
  2. 提出相似性推理模块来利用行人关系来优化推理。当查询图像 q 与图库图像 g 表现出高度相似性时，适合减小 q 与其他类似 g 图像之间的距离。我们的方法用亲和力矩阵模拟上述过程来修正距离计算。

3.2 Semantic-Aligned Feature Learning

为了缓解行人运动引起的部分特征错位问题，我们提出了语义对齐特征学习。核心思想是使用learnable prototypes来提取隐藏的语义特征从Fr∈Rh×w×cF_r\in\mathbb{R}^{h\times w\times c}Fr​∈Rh×w×c和Fv∈Rh×w×cF_v\in\mathbb{R}^{h\times w\times c}Fv​∈Rh×w×c

具体来说，当从特征图中提取潜在语义部分特征时，我们应该为与目标部分相似的像素级特征赋予更高的权重。该方法可以形成局部的一致性，保证局部特征的一致性和可靠性。

但我们没有每个部分的参考特征。因此，我们建议利用可学习原型 P=[P1,P2,⋯ ,Pp]∈Rp×cP=[P_1,P_2,\cdots,P_p]\in\mathbb{R}^{p\times c}P=[P1​,P2​,⋯,Pp​]∈Rp×c 作为潜在语义部分特征的参考，其中 Pi 表示第 i 个可学习原型，p 表示此类可学习原型的总数可学习的原型。我们利用可学习的原型来学习适当的潜在语义部分特征作为参考。每个可学习原型的目标是从特征图中聚集独特的潜在语义部分特征。

为了实现两种模态之间零件特征的语义对齐，我们对可见光模态和红外模态采用共享原型 P。共享原型能够将两种模态中具有相同语义信息的潜在语义部分特征聚合到相同的部分中。这种设计可以实现潜在语义部分级别的模态对齐，减少模态差异。当提取潜在语义部分特征时，两种模态的像素级特征将与共享的可学习原型进行比较，以确定它们对每个潜在语义部分特征的加权贡献。为了描述方便，我们以可见特征为例来描述细节。

3.3 Affinity Inference Module

基于欧氏距离或余弦距离匹配行人图像的有效性是有限的。这些方法忽略了相似性信息，并在推理过程中将图库图像视为不同的实体。因此，我们提出了亲和力推理模块（AIM），它利用行人关系来修正距离。

AIM 背后的直觉是，我们可以利用与查询图像具有高亲和力的图库图像来修改距离。 AIM可以捕获图库图像之间潜在的亲和力信息并将其纳入距离计算中，优化匹配性能。算法 1 概述了 AIM 的主要步骤。

创新点

我们提出了用于可见红外人再识别的语义对齐和相似性推理框架（SAAI）。我们首先设计了一种语义对齐的特征学习过程，通过考虑像素级特征和可学习原型之间的相似性来对齐潜在语义部分。该方法有效降低了行人运动的影响，增强了特征表示的独特性。此外，我们提出了一个亲和力推断模块，它利用行人关系来修改距离计算。通过在 SYSU-MM01 和 RegDB 数据集上进行的广泛实验，我们的框架展现了 VI-ReID 的显着功效。

局限性和未来的工作

我们的框架仅在推理阶段利用行人关系。然而，亲和力信息在训练阶段仍未得到利用，这阻碍了训练过程的进展。我们未来的工作是将我们的亲和力推理方法与网络训练相结合。
