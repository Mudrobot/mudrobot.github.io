文章内容

Abstract

Most video-based person re-identiﬁcation (re-id) methods only focus on appearance features but neglect motion features. In fact, motion features can help to distinguish the target persons that are hard to be identiﬁed only by appearance features. However, most existing temporal information modeling methods cannot extract motion features effectively or efﬁciently for v ideo-based re-id. In this paper, we propose a more efﬁcient Motion Feature Aggregation (MFA) method to model and aggregate motion information in the feature map level for videobased re-id. The proposed MFA consists of (i) a coarse-grained motion learning module, which extracts coarse-grained motion features based on the position changes of body parts over time, and (ii) a ﬁne-grained motion learning module, which extracts ﬁne-grained motion features based on the appearance changes of body parts over time. These two modules can model motion information from different granularities and are complementary to each other. It is easy to combine the proposed method with existing network architectures for end-to-end training. Extensive experiments on four widely used datasets demonstrate that the motion features extracted by MFA are crucial complements to appearance features for video-based re-id, especially for the scenario with large appearance changes. Besides, the results on LS-VID, the current largest publicly available video-based re-id dataset, surpass the state-of-the-art methods by a large margin. The code is available at: https://github.com/guxinqian/SimpleReID.

大多数基于视频的行人重识别（re-id）方法只关注外观特征而忽略运动特征。事实上，运动特征可以帮助区分仅通过外观特征难以识别的目标人物。然而，大多数现有的时间信息建模方法不能有效或高效地提取运动特征以用于基于视频的重识别。在本文中，我们提出了一种更有效的运动特征聚合（MFA）方法来建模和聚合基于视频的重识别的特征图级别的运动信息。所提出的MFA由（i）粗粒度运动学习模块组成，该模块根据身体部位随时间的位置变化提取粗粒度运动特征，以及（ii）细粒度运动学习模块，该模块提取细粒度运动特征。基于身体部位随时间的外观变化的颗粒化运动特征。这两个模块可以对不同粒度的运动信息进行建模，并且相互补充。很容易将所提出的方法与现有网络架构相结合以进行端到端训练。对四个广泛使用的数据集的大量实验表明，MFA 提取的运动特征是基于视频的重识别的外观特征的重要补充，特别是对于外观变化较大的场景。此外，目前最大的公开的基于视频的重识别数据集 LS-VID 的结果大大超过了最先进的方法。代码位于：https://github.com/guxinqian/SimpleReID.

总结

之前工作存在的问题

ReID中提取行人姿势的变化：

  • 基于RNN的工作，通常将池化后的CNN特征作为输入，身体部位特征的空间位置由于池化大量丢失
  • 虽然基于C3D的方法[23]、[24]可以通过3D卷积对特征的相对位置进行编码，但当存在时间外观未对齐时，它们可能会混淆不同的外观特征[12]，这对重新识别是有害的。
  • 基于流的方法可以有效地根据连续帧的光流对运动进行建模。光流的提取和建模需要额外的模型，并且非常耗时。

反思

  • 基于 RNN 的方法 [10]、[20] 通常将池化操作后的 CNN 特征馈送到循环模型中，例如 LSTM [53] 或 GRU [54]，以对多个帧之间的时间关系进行建模。由于池化操作，身体部位特征的空间信息大量丢失，因此在此过程中运动信息可能被破坏。在[21]中，Zhang 等人。表明对 RNN 输入序列的时间顺序进行打乱可以获得比原始顺序更好的重识别性能。所以RNN带来的性能提升可能不是因为它能够提取顺序运动特征，而是因为外观特征增强带来的。
  • 基于图的方法[16]、[17]将时空特征图中不同身体部位或所有像素的特征视为图的节点，并将它们输入GCN [55]来建模时空关系。类似于基于 Transformer [56] 的方法然而，它们不会对特征的时间位置进行编码，并将 GCN 的输入视为一个集合而不是一个序列。因此，用这些方法很难提取运动特征。
  • 基于 C3D 的方法 [24]、[50]、[57]-[60] 可以通过级联多个 3D 卷积来编码相对位置和模型运动信息 [23]，但当存在时间外观未对齐时，它们可能会混淆外观表示 [12] ]。尽管顾等人。 [12]通过在3D卷积之前对齐外观特征来缓解这个问题，但由于外观对齐导致空间位置丢失，运动信息仍然无法很好地保持。
  • 基于流的方法[51]、[52]可以分别从原始图像和光流中提取外观特征和运动特征，并将它们融合以充分利用时空信息。然而，光流的提取和建模需要额外的模型并且非常耗时。这对于实时应用来说是非常不利的。

主要贡献

  1. 反思现有时间信息建模方法的有效性和效率。是ReID领域关于运动特征提取 的一篇文章
  2. 提出MFA方法，可以有效地提取特征图级别的运动特征；
  3. 证明MFA提取的运动特征作为外观特征的重要补充的有效性，特别是对于一些困难的情况；
  4. 与最先进的方法相比，在广泛使用的基于视频的重识别数据集上实现了卓越的性能。

关键就是其中的Coarse-Grained Motion Learning (CGML module) 和 Fine-Grained Motion Learning (FGML module)。它们可以从不同的粒度对特征图中的时间信息进行建模，并且彼此互补。通过残差连接很容易将这两个模块插入到大多数现有的神经网络中。在这种情况下，集成网络可以同时对外观和运动表示进行建模。由于很难同时学习两个残差项，因此我们以顺序方式集成这两个模块，并将它们分别插入到不同的残差块之后。这两个模块的详细信息将在接下来的两小节中介绍。

CGML 讨论

CGML中计算的偏移量与光流类似，但它们之间有很多差异。

  1. 计算复杂度和时间消耗。基于流的方法需要两个额外的网络来分别提取和建模光流。相比之下，CGML 中的偏移量计算是无参数的，并且 CGML 中的偏移量建模是轻量级的（只有四个卷积层）。因此，CGML的计算复杂度和时间消耗都低于基于流的方法。
  2. 模型训练。 CGML 中偏移量的计算是可微分的，并且很容易将 CGML 与现有网络架构结合起来进行端到端训练，而无需额外的数据和监督。相比之下，基于流的方法中的光流提取网络（例如 RAFT [13] 和 FlowNet [25]）需要使用额外的带注释的流数据（例如 Sintel [26] 和 KITTI [14]）单独进行预训练。 ]）。
  3. 当预训练的流数据集和re-id数据集之间的域差距很大时，估计的光流通常包含许多噪声（见图1），而CGML可以在re-id数据集上学习更精确的运动特征（见图8）。至于基于视频的重识别的准确性，CGML也优于基于流的方法（如表5所示）。

Fine-Grained Motion Learning

由于中级特征的感受野较大，中级特征图中的一个像素可以捕获原始图像中某个区域（对应身体部位）的外观信息。虽然CGML可以提取粗粒度的运动特征来捕获身体部位的剧烈变化，但它无法捕获相对较小区域内的细粒度的身体运动。作为补救措施，我们提出 FGML 来提取细粒度的运动信息。

同上，我们进行如下定义：

Δfit=fjt+Δt−fit\Delta f_i^t = f_j^{t+\Delta t}-f_i^t Δfit​=fjt+Δt​−fit​

同理fjt+Δtf_j^{t+\Delta t}fjt+Δt​是i在下一帧对应的特征位置

还是采用:

fjt+Δt=∑kes⋅cos⁡(fit,fkt+Δt)fkt+Δt∑k′es⋅cos⁡(fit,fk′t+Δt)f_j^{t+\Delta t}= \sum_k\frac{e^{s\cdot\cos(f_i^t,f_k^{t+\Delta t})}f_k^{t+\Delta t}}{\sum\limits_{k'}e^{s\cdot\cos(f_i^t,f_{k'}^{t+\Delta t})}} fjt+Δt​=k∑​k′∑​es⋅cos(fit​,fk′t+Δt​)es⋅cos(fit​,fkt+Δt​)fkt+Δt​​

在计算了外观特征在时间维度上的差异后，我们使用 1 × 1 卷积，然后使用 BatchNorm [64] 层将它们转换为细粒度的运动特征。该卷积的输入通道数和输出通道数均为C，因此细粒度运动特征的大小与输入特征图的大小相同。进行残差连接很方便。 FGML模块的详细架构如图6所示。
